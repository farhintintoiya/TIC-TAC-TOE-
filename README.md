

# Tic-Tac-Toe Game in C

## Overview

This is a **console-based Tic-Tac-Toe game** developed in C. Two players can play on the same computer by taking turns. The game checks for wins and draws automatically and displays the updated board after every move.

---

## Features

* Two-player gameplay (Player X and Player O).
* Displays a 3x3 board with updated moves.
* Validates input to prevent overwriting existing moves or entering invalid positions.
* Detects win or draw conditions and announces the result.
* Minimalistic and easy-to-understand console interface.

---

## Technologies Used

* **Language:** C
* **Concepts:** Arrays, Functions, Loops, Conditionals, Pointers (basic), Console Input/Output
